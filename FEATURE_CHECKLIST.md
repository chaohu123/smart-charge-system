## 系统架构

### 后端技术栈
- Spring Boot
- MyBatis Plus
- MySQL
- Redis（配置中）
- WebSocket（实时通信）
- JWT（认证）

### 前端技术栈
- Vue 3
- Element Plus
- Vite
- Axios
- WebSocket客户端

### 数据库
- MySQL 8.0+
- 字符集：utf8mb4
- 时区：Asia/Shanghai

## 部署说明

### 后端部署
1. 配置数据库连接（application.yml）
2. 执行数据库初始化脚本（init.sql）
3. 启动Spring Boot应用（端口8080）

### 前端部署
1. 用户端：端口3000
2. 管理员端：端口3001
3. 配置API代理（vite.config.js）
