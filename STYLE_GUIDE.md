# 样式设计指南

## 概述

本项目采用统一的设计系统，包含用户端和管理员端两套样式方案，确保界面美观、一致且易于使用。

## 设计原则

1. **一致性**：保持视觉风格和交互模式的一致性
2. **可访问性**：确保良好的对比度和可读性
3. **响应式**：适配不同屏幕尺寸
4. **性能**：优化动画和过渡效果

## 通用设计模式

### 1. 卡片设计

所有卡片遵循以下规范：
- 圆角：12px（大卡片）或 8px（小卡片）
- 阴影：轻微阴影，悬浮时增强
- 内边距：16px-24px
- 过渡：0.3s ease

### 2. 按钮设计

- **主要按钮**：渐变背景，白色文字
- **次要按钮**：边框样式，主色调文字
- **文字按钮**：无背景，主色调文字
- **图标按钮**：圆形，图标居中

### 3. 输入框设计

- 边框：1px 实线，默认灰色
- 聚焦：边框变为主色调
- 圆角：8px
- 内边距：12px

### 4. 动画效果

#### 淡入动画
```css
.fade-in {
  animation: fadeIn 0.3s ease-out;
}
```

#### 滑入动画
```css
.slide-in {
  animation: slideIn 0.3s ease-out;
}
```

#### 缩放动画
```css
.scale-in {
  animation: scaleIn 0.3s ease-out;
}
```

### 5. 状态指示器

充电状态使用脉冲动画：
```css
.status-indicator.charging {
  animation: pulse 2s infinite;
}
```

## 最佳实践

### 1. 使用CSS变量

优先使用CSS变量而非硬编码颜色：
```css
/* ✅ 正确 */
color: var(--primary-color);

/* ❌ 错误 */
color: #00C853;
```

### 2. 保持间距一致

使用预定义的间距变量：
```css
padding: var(--spacing-md);
margin-bottom: var(--spacing-lg);
```

### 3. 使用过渡效果

为交互元素添加过渡：
```css
transition: all var(--transition-base);
```

### 4. 响应式优先

使用移动端优先的响应式设计：
```css
/* 基础样式（移动端） */
.card {
  padding: var(--spacing-sm);
}

/* 桌面端增强 */
@media (min-width: 768px) {
  .card {
    padding: var(--spacing-lg);
  }
}
```

### 用户端

- **主色**：用于主要操作按钮、链接、强调内容
- **辅助色**：用于次要信息、图标
- **强调色**：用于价格、重要提示

- **警告色**：用于警告提示
- **危险色**：用于错误、禁用状态

### 管理员端

- **主色**：用于主要操作、标题
- **成功色**：用于正常状态、完成状态
- **警告色**：用于待处理状态
- **危险色**：用于错误、故障状态
- **信息色**：用于提示信息

## 字体规范

### 字体大小

- **超大标题**：24px
- **大标题**：20px
- **标题**：18px
- **正文**：16px
- **小字**：14px
- **极小字**：12px

### 字体粗细

- **标题**：600（semi-bold）
- **正文**：400（normal）
- **强调**：500（medium）

## 阴影规范

- **小阴影**：0 2px 4px rgba(0, 0, 0, 0.08)
- **中阴影**：0 4px 12px rgba(0, 0, 0, 0.12)
- **大阴影**：0 8px 24px rgba(0, 0, 0, 0.16)

## 圆角规范

- **小圆角**：4px（标签、小按钮）
- **中圆角**：8px（输入框、卡片）
- **大圆角**：12px（大卡片）
- **超大圆角**：16px（特殊卡片）

## 性能优化

1. **使用transform而非position**：动画性能更好
2. **避免频繁重排**：使用transform和opacity
3. **合理使用will-change**：仅用于动画元素
4. **减少阴影层级**：避免过度使用阴影

## 浏览器兼容性

- Chrome/Edge：最新版本
- Firefox：最新版本
- Safari：最新版本
- 移动端浏览器：iOS Safari、Chrome Mobile
